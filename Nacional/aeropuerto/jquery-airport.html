jQuery Airport

Lightweight Text Effect. (0.7kb)

First d

HTML:
Javascript:
$(".example1").airport([ "First word", "2nd Word", "And third" ]);

The simplest way to use jQuery Airport is to just call the function on your selected element and pass it an array of words as shown above. The element can contain a word or just be left empty.

CSS:
(prefixes left out for brevity) 
.example2 {    transition: color 0.5s ease-in;}
Javascript:
var words = [ "Colors", "can", "be", "really really cool" ];var colors = "#AAFF12, #FF9712, #12FFF6, #FF1271, #FFEC12";$(".example2").airport(words, {    fill_space: true,    colors: colors});

I've found that when using airport with text that is centered, the fill_space option might provide unwated results. You can see from this example the fill_space option fills in all the spaces of the previous word. By using a combination of the provided colors string and a little bit of css magic, we can achieve a smooth fade between color changes.

Options

transition_speed: [Number=1000]

The time in ms before switching to the next word.

loop: [Boolean=true]

Choose whether to stop at the end of your array values or keep looping through.

fill_space: [Boolean=false]

Puts spaces (nbsp) after shorter words within your list to fill in space of your longest word. (If your words are "One, Two, Three," "One" and "Two" will be written as "One  " and "Two  ")

colors: [String=null]

List of comma separated colors to randomly color your values i.e. '#b5c523,#E7860E'
